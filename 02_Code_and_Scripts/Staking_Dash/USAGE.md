# Staking Dash – Usage

This folder hosts code and notes relevant to the Staking Dash (Power BI) solution.

## Suggested Structure
- `src/` – parameterized Power Query M snippets, DAX exports, or helper scripts.
- `data/` – sample/mock datasets (no PII or confidential data).

## Notes
- Do not commit real credentials or internal endpoints.
- Include PBIX via release assets or keep locally (PBIX often large).
